# SHT45_Filter_Board
SHT45-SXG模組篩選板
### 主要功能:  
* 同時讀取30pcs 的SHT45-SXG溫濕度模組
* 上傳數據到 雲端網頁
* 人工篩選出 特性、靈敏度一致的模組
